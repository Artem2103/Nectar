## Phase 1 — Data Layer  (foundation; everything else depends on this)

- [x] Add packages: supabase_flutter, image_picker, http, uuid
- [x] MealEntry model  (id, timestamp, name, imagePath, kcal, protein, carbs, fat, fiber, sugar, sodium)
- [x] WeightEntry model  (id, timestamp, valueKg)
- [x] UserGoals model  (dailyKcal, proteinG, carbsG, fatG, startWeightKg, goalWeightKg)
- [x] MealRepository  — CRUD, persisted to Supabase `meal_entries` (RLS-scoped per user)
- [x] WeightRepository — CRUD, persisted to Supabase `weight_entries` (RLS-scoped per user)
- [x] GoalsRepository  — read/upsert UserGoals to Supabase `user_goals` (RLS-scoped per user)
- [x] Riverpod providers for each repository (AsyncNotifierProvider)
- [x] dailySummaryProvider: read today's meals from MealRepository, compute remaining macros vs UserGoals
- [x] progressSummaryProvider: read WeightRepository + MealRepository (streak logic)
